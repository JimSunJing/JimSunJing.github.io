StringBuffer, StringBuilder 差异

Java 提供了三种代表字符序列的类: String, StringBuffer, StringBuilder. String是不可变类型，而StringBuffer, StringBuilder都是可变类型。StringBuffer和StringBuilder有很多差别，StringBuilder是自JDK1.5新增的的。

主要的差别如下表：

No. StringBuffer StringBuilder
1) StringBuffer 是同步化的，也就是多线程安全的。这意味着两个Thread不可以同时调用StringBuffer的方法。 StringBuilder 非同步化的，多线程不安全。两个线程可以同时调用StringBuilder的方法。
2) StringBuffer 比 StringBuilder 效率低。 StringBuilder 比 StringBuffer 更高效。

源码

StringBuffer 和 StringBuilder 都是继承了 AbstractStringBuilder 类。在 StringBuffer 中大多数方法都被 synchronized 修饰，所以保障了线程安全。
