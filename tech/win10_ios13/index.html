Windows 电脑与 iPhone/iPad 在同一Wifi下的文件(影视)共享 (ios13 only)

Windows端设置与部署

该部分为搬运 Charles 的知乎文章

  • 在电脑任意位置新建文件夹
  • 右键文件夹选择“属性”
  • 点击最顶上第二个“共享”
  • 点击“高级共享“，勾选“共享此文件夹”

  • 不要关掉上面那个“高级共享”的窗口
  • 点击下面的“权限”，然后全部勾选“允许”

  • 回到刚才那个“共享”标签，点击“共享”
  • 在下拉菜单里选择“Everyone”，点击右边的“添加”按钮
  • 记得把“权限级别”改为“读取/写入”

  • 使用快捷键 Windows键+R 打开 运行，输入CMD，回车
  • 在命令行中输入 ipconfig 点击回车。记住User/后面的这几个字母，记住IPv4地址

得到结果：

  1. 你的Windows用户名 username (CMD图片中的"imcha")
  2. 你连接wifi的IPv4地址 ipv4 (CMD图片中的“192.168.0.26)
  3. 你的windows账号密码 password
    注意：不是你的开机密码，是买电脑后初次注册的windows账号的密码。
    windows账户密码忘记重置

iOS13使用系统文件app自带smb连接共享文件夹

  • 打开 iOS 13 自带的的文件App
  • 点击右上角的三个小点
  • 点击 Connet to server（中文是“连接服务器”）
  • Server 里面输入 smb://ipv4地址 比如：

smb://192.168.0.26 （注意 smb小写！！！）

  • 选择Registered User（注册用户）

  • 在Name输入上面那张图User/后面的这几个字母username

  • Password就是你微软账号的密码password

  • 点击Next既可成功连接你的电脑文件夹啦！

使用 VLC app 直接解析观看分享文件夹中的电影/视频

VLC可以借助smb连接共享文件夹直接观看视频。

  • 进入网络一栏
  • 点击文件服务器的那个连接
  • 选择 SMB 连接
  • 输入你的ipv4地址，端口为空，windows用户名username以及密码password
  • 点击保存，方便下次连接
  • 再点击右上角的”连接“
  • 进入文件管理系统，可以访问你已经共享的文件夹了
  • 点击视频或音乐文件可以直接打开看，很流畅。但是拖动进度条可能需要缓存一下。
